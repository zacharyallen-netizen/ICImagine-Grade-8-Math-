Special Cases: No Solution & Infinite Solutions
💡 Key: Variables cancel and you get a TRUE statement (like 5=5) → Infinite solutions. FALSE statement (like 3=7) → No solution.

🔍 Common Mistakes

Can you spot what went wrong?

Solve: −4x > 20
✗ Student: Student writes: x > −5
✓ Fix: WRONG! Must FLIP the inequality when dividing by negative. Answer: x < −5
💡 ALWAYS flip < > ≤ ≥ when multiplying or dividing by a negative.
Solve: 2x + 3 = 2x + 7
✗ Student: Student writes: x = 2
✓ Fix: WRONG! Subtract 2x from both sides: 3 = 7 (FALSE) → No Solution
💡 If variables cancel and you get a false statement, there's no solution.

⚡ Key Tips for This Unit

✅ FLIP the inequality when you multiply or divide by a NEGATIVE.
✅ No solution: variables cancel → false statement (3 = 7).
✅ Infinite solutions: variables cancel → true statement (5 = 5).
✅ Always check by substituting back into the ORIGINAL equation.
