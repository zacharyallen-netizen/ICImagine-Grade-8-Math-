① What Makes a Table Linear?

🔑 Key Idea

A function is linear if it has a constant rate of change — the slope (Δy ÷ Δx) is the same between every pair of consecutive points.

We check this by adding change columns to both sides of the table.

How to Read the Change Columns

  1. Δx column (left side) — subtract consecutive x-values going down.
  2. Δy column (right side) — subtract consecutive y-values going down.
  3. If Δy ÷ Δx is the same every time → the function is linear, and that ratio is the slope.
Example 1: Is this table linear?
ΔxxyΔyΔy/Δx
13
+125+22/1 = 2
+137+22/1 = 2
+149+22/1 = 2

✓ LINEAR — Every Δy/Δx = 2. The slope is 2 and the equation is y = 2x + 1.

Example 2: Is this table linear?
ΔxxyΔyΔy/Δx
01
+112+11/1 = 1
+125+33/1 = 3
+1310+55/1 = 5

✗ NOT LINEAR — The slopes are different (1, 3, 5). The rate of change is not constant.

② Linear vs Non-Linear: Side by Side

✓ LINEAR

+221−3−3/2
+24−2−3−3/2
+26−5−3−3/2

Constant slope = −3/2
Equation: y = −3/2 x + 4

✗ NON-LINEAR
ΔxxyΔyΔy/Δx
11
+124+33/1 = 3
+139+55/1 = 5
+1416+77/1 = 7

Slopes change (3, 5, 7)
This is y = x² — a quadratic!

⚡ Quick Test

If Δx is constant (same gap between every x), then you only need to check if Δy is also constant.

If Δx is NOT constant, you must compute Δy/Δx for each interval to check the slope. That's the tricky case!

③ The Tricky Case: Uneven X-Spacing

Watch out! When the x-values aren't evenly spaced, the changes in y will look different — but the function can still be linear. You have to divide Δy by Δx every time to check the actual slope.

Example 3: Don't be fooled — check the slope!

At first glance, Δy keeps changing (6, 9, 12). Looks non-linear... but look at Δy/Δx!

ΔxxyΔyΔy/Δx
01
+227+66/2 = 3
+3516+99/3 = 3
+4928+1212/4 = 3

✓ LINEAR! Even though Δx and Δy both change, the slope is constant at 3. The equation is y = 3x + 1.

Lesson: Always compute Δy ÷ Δx — don't just eyeball Δy alone!

Example 4: Another tricky one — is this linear?
ΔxxyΔyΔy/Δx
15
+231−4−4/2 = −2
+58−9−10−10/5 = −2
+311−15−6−6/3 = −2

✓ LINEAR! Slope = −2 every time. Equation: y = −2x + 7
