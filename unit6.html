Introduction to Functions

NC.8.F.1 — Understand that a function assigns each input exactly one output

What is a Function?
DEFINITIONA function is a relation in which every input (x) has exactly one output (y). Each input maps to one and only one output.

⭐ The One-Output Rule

If ANY input (x) maps to MORE than one output (y), it is NOT a function!

Four Representations

Ordered Pairs

{(1,3), (2,5), (3,7)}
Pairs of (input, output)

Mapping Diagram

Table

y column = outputs

❌ NOT a Function

{(0, 3), (1, 4), (2, 8), (1, 6)}

The x-value 1 appears twice with different y's (4 and 6). ✗

Vertical Line Test (VLT)

For graphs, we use the Vertical Line Test: draw a vertical line anywhere on the graph. If it ever crosses the graph in two or more places, the relation is NOT a function.

💡 Quick Memory Trick

"One output per vertical column" — if a vertical line hits the graph twice, two y-values share one x-value = NOT a function.
Function Notation
f(x) = 2x + 4

Read as "f of x equals 2x + 4"

f(3) = 10

f(3) means: find the output when x = 3

📝 Evaluating a Function

Given f(x) = 3x − 3, find f(2).

Substitute 2 in for x:   f(2) = 3(2) − 3

Multiply:   f(2) = 6 − 3

f(2) = 3 ✓

🚫 NOT Functions as Equations

If y appears in the denominator OR if the equation has y², it is NOT a function. Vertical lines (x = 6) are NOT functions. Horizontal lines (y = 7) ARE functions.
Equation Examples

An equation is a function if every x-value produces exactly one y-value. Here are some to compare:

✅ FUNCTIONS

y = 3x + 5

Linear — one y per x. ✓

y = x² + 1

Curved, but each x still gives one y. ✓

y = 7

Horizontal line — every x gives the same y = 7. ✓

❌ NOT FUNCTIONS

x = 4

Vertical line — one x has infinite y's. ✗

y² = x

Has y² — each x gives two y's (e.g., x = 9 → y = 3 or −3). ✗

x² + y² = 25

Circle — most x-values give two y's. ✗

💡 Quick Check

If the equation can be solved for y in the form y = (something with x) and that "something" gives only one answer, it's a function. Watch out for y², y in the denominator, or x = a number — those are red flags.

6.2

Linear Functions & Slope

NC.8.F.3 — Interpret y = mx + b; NC.8.F.4 — Determine rate of change

What Makes a Function Linear?

⭐ Linear Function

A linear function has a constant rate of change and produces a straight line when graphed (not horizontal or vertical).

We use the term slope to describe the rate of change — how steeply the line rises or falls.

m =ΔyΔx=y₂ − y₁x₂ − x₁

Slope = Rise over Run  ·  Pick any two points (x₁,y₁) and (x₂,y₂)

Finding Slope

📝 Example 1: Find slope from two points

Find the slope through (2, 5) and (6, 13).

Label the points: (x₁, y₁) = (2, 5) and (x₂, y₂) = (6, 13)

Apply the formula:

13 − 56 − 2=84

m = 2 ✓

📝 Example 2: Find slope from a graph

Count the rise (vertical change) over the run (horizontal change) between two clear lattice points.

Pick two clear lattice points on the line: (0, −1) and (3, 2).

Count the rise (vertical change): go up 3 units.

Count the run (horizontal change): go right 3 units.

m = riserun = 33

m = 1 ✓

Slope-Intercept Form
y = mx + b

m = slope (rate of change)  ·  b = y-intercept (where line crosses the y-axis)

📝 Example: Graph using slope-intercept form

Graph y = 2x − 1

  • Identify b = −1. Plot the y-intercept at (0, −1).
  • Identify m = 2 = 21. From (0,−1), go UP 2, RIGHT 1 → plot (1, 1).
  • Draw a line through the two points and extend in both directions.

📝 Example: Write an equation from a graph

To write y = mx + b from a graph, find b (where the line crosses the y-axis) and m (rise over run).

Step 1 — Find b: the line crosses the y-axis at (0, 2), so b = 2.

Step 2 — Find m: from (0, 2) to (2, 0), the line goes down 2 (rise = −2) and right 2 (run = +2).

m = −22 = −1

Step 3 — Write the equation: plug m = −1 and b = 2 into y = mx + b.

y = −x + 2 ✓

Four Types of Slope

POSITIVE ↗

Rises left to right
m = positive number

NEGATIVE ↘

Falls left to right
m = negative number

ZERO →

Perfectly horizontal
m = 0

UNDEFINED ↕

Perfectly vertical
m = undefined

Writing Equations from Tables

NC.8.F.4 — Construct a function; determine rate of change from tables

Is a Table Linear?

To check if a table is linear, track the change in x (Δx) and the change in y (Δy) between every pair of consecutive rows, then calculate the slope Δy ÷ Δx. If the slope is always the same, the relationship is linear.

⚠️ Important!

Always check slope (Δy ÷ Δx), NOT just Δy alone — x-values may not always increase by the same amount!

✅ LINEAR — slope is constant

ΔxxyΔyΔy ÷ Δx

❌ NON-LINEAR — slope changes

Slope changes (1, 2, 4, 8) → NOT LINEAR ✗

⚠️ Don't Be Fooled — Always Check the Slope!

Sometimes a table has inconsistent x-intervals, so the y-values jump around in ways that look non-linear at first glance. You must calculate Δy ÷ Δx for every pair before you can decide!

📝 Example: Looks non-linear... but IS linear

At first glance, the Δy values (+4, +12, +8, +20) look wildly inconsistent. But watch what happens when we divide by Δx:

ΔxxyΔyΔy ÷ Δx
—15——
+129+44 ÷ 1 = 4
+3521+1212 ÷ 3 = 4
+2729+88 ÷ 2 = 4
+51249+2020 ÷ 5 = 4

The Δy column looks chaotic — but the x-values don't jump by the same amount each time.

Once you divide Δy ÷ Δx, every slope equals 4.

This table IS linear! ✓ Slope = 4

💡 The Big Takeaway

Never decide a table is non-linear just by looking at Δy. Always compute Δy ÷ Δx for every pair — only then can you compare slopes fairly.
Steps to Write the Equation
  • Find slope: build a table with Δx and Δy columns → m = Δy ÷ Δx
  • Find the y-intercept: use the row where x = 0. If no x = 0 row, either extend the pattern back to x = 0, or substitute m and a point into y = mx + b and solve for b algebraically.
  • Write the equation in slope-intercept form: y = mx + b

Step 2 — Y-intercept: look at the x = 0 row → y = 2, so b = 2

Equation: y = 3x + 2 ✓

📝 Example 2: No x = 0 row — use the pattern OR algebra

Write an equation for the table below. Notice there is no row where x = 0, so the y-intercept is not handed to us.

ΔxxyΔyΔy ÷ Δx
—27——

Step 1 — Slope: every Δy ÷ Δx = 4, so m = 4

Option A — Extend the pattern backward to x = 0

The y-values go UP by 4 every time x goes UP by 1. To get to x = 0, we need to go backward: subtract 1 from x and subtract 4 from y.

ΔxxyΔy
—0−1—
+113+4
+127+4
+1311+4

The highlighted row shows x = 0, y = −1, so b = −1.

Equation: y = 4x − 1 ✓

Option B — Solve for b algebraically

Pick any point from the table — say (2, 7) — and substitute into y = mx + b with m = 4:

Start with:   y = mx + b

Substitute m = 4 and (x, y) = (2, 7):   7 = 4(2) + b

Multiply:   7 = 8 + b

Subtract 8 from both sides:   b = −1

Equation: y = 4x − 1 ✓ (same answer!)

💡 Which method should you use?

Both give the same answer. Extending the pattern is fast when the slope is a whole number and x = 0 is close. Algebra is more reliable when the slope is a fraction or when x = 0 is far away. Pick whichever feels cleaner for the table in front of you.

6.4

Real-World Context

NC.8.F.4 — Interpret rate of change and initial value in context

Translating Words to Equations
per / each / every→This is the slope (m) — the rate of change
flat fee / starts with / initial→This is the y-intercept (b) — the starting value
base fee / one-time charge→This is also the y-intercept (b)

Slope (m) tells you:

"For every 1 more [x-unit], the [y quantity] changes by [m]."

Y-intercept (b) tells you:

"When [x] equals zero, [y] is [b]."

📝 Example 1: Taxi Fare

A taxi charges a $3 flat fee plus $2.50 per mile. How much for 7 miles?

Slope: $2.50 per mile → m = 2.50

Y-intercept: $3 flat fee → b = 3

Equation: y = 2.50x + 3

7-mile ride: y = 2.50(7) + 3 = $20.50 ✓

📝 Example 2: Car Value

y = −1500x + 24000 models a car's value after x years.

Slope (−1500): the car loses $1,500 in value every year

Y-intercept (24000): the car was worth $24,000 when new (x = 0)

After 5 years: y = −1500(5) + 24000 = $16,500 ✓

📝 Example 3: Gym Membership

A gym charges a $40 sign-up fee plus $25 per month. Write an equation for the total cost y after x months, then find the cost after 8 months.

Y-intercept: $40 sign-up fee (paid at x = 0) → b = 40

Slope: $25 per month → m = 25

Equation: y = 25x + 40

After 8 months: y = 25(8) + 40 = $240 ✓

Qualitative Graphs

NC.8.F.5 — Describe functional relationships; sketch from verbal description

Key Vocabulary
Increasing ↗
Line goes UP left to right. Output grows as input grows.
Decreasing ↘
Line goes DOWN left to right. Output shrinks as input grows.
Constant →
Flat horizontal line. Output stays the same.
Linear
Straight line — constant slope throughout.
Non-linear
Curved line — slope is changing.
Steep
Sharp angle — fast rate of change.

⭐ Remember

Faster = steeper  ·  Slower = shallower  ·  Stopped = flat horizontal
Reading & Sketching Graphs

Steps to READ a graph:

  • Look at each section separately, moving left to right.
  • For each section: going up, down, or flat?
  • Is it a straight line (linear) or a curve (non-linear)?
  • Is the slope steep or shallow?

Steps to SKETCH from a story:

  • Break the story into phases (usually 2–4 phases).
  • For each phase: does the quantity increase, decrease, or stay flat?
  • Decide: steady change (straight line) or speeding up/slowing down (curve)?

📝 Example: Swimmer's Distance-Time Graph

PhaseWhat HappensGraph Feature
Swimming to far wallDistance increases steadilyStraight line going UP
Resting at wallDistance stays the sameFLAT horizontal line
Swimming back fasterDistance decreases fasterSteeper line going DOWN

The return leg is steeper than the outbound leg because the swimmer is moving faster.

📝 Example: Cell Phone Battery Over a School Day

A student starts the day with a full battery. During 1st block the phone is in her locker (battery doesn't change). During lunch she scrolls social media (drops fast). During afternoon classes the phone is off (battery doesn't change). After school she watches videos (drops fast again until dead).

Systems of Equations

NC.8.EE.8 — Solve systems of two linear equations

What is a System?
DEFINITIONA system of equations is a set of two or more equations with the same variables. The solution is the (x, y) coordinate where both equations are true at the same time.

ONE Solution

Lines intersect at one point. Different slopes.

NO Solution

Parallel lines. Same slope, different b.

INFINITE Solutions

Same line. Same slope and same b.

Method 1: Inspection
What You NoticeResultExample
Different slopesONE solutiony = 3x + 2 and y = −x + 6
Same slope, different bNO solutiony = 3x + 2 and y = 3x − 5
Identical equationsINFINITE solutionsy = 3x + 2 and 2y = 6x + 4
Method 2: Graphing
  • Graph both equations on the same coordinate plane (use y = mx + b: plot b, then use slope to find a second point).
  • Find the point where the two lines intersect.
  • Read the (x, y) coordinates of that point — that's your solution.
  • Check your answer by substituting x and y into BOTH equations.

📝 Example: Solve by graphing — y = x + 1 and y = −x + 5

Graph y = x + 1: b = 1, so plot (0, 1). Slope = 1 = 11 → up 1, right 1.

Graph y = −x + 5: b = 5, so plot (0, 5). Slope = −1 → down 1, right 1.

Find the intersection: the two lines cross at (2, 3).

Solution: (2, 3) ✓

🔍 Why is (2, 3) the valid solution?

The intersection point is the ONE ordered pair that lies on both lines at the same time. That means x = 2 and y = 3 must make BOTH equations true. Let's check:

Check in equation 1: y = x + 1 → 3 = 2 + 1 → 3 = 3 ✓

Check in equation 2: y = −x + 5 → 3 = −(2) + 5 → 3 = 3 ✓

Both equations are true at (2, 3), so it's a valid solution. Any other point would only be on one line — not both — so it couldn't solve the system.

Method 3: Substitution
  • Rewrite one equation so a variable is isolated (y = … or x = …)
  • Substitute that expression into the OTHER equation.
  • Solve for the remaining variable.
  • Substitute that value back in to find the other variable.
  • Write your answer as an ordered pair (x, y). Check in BOTH equations!

📝 Example: y = 4x − 1 and y = 2x − 5

Since both equal y:   4x − 1 = 2x − 5

Subtract 2x from both sides:   2x − 1 = −5

Add 1 to both sides:   2x = −4  →  x = −2

Substitute x = −2 into y = 4x − 1:   y = 4(−2) − 1 = −9

Solution: (−2, −9) ✓   Check: 2(−2)−5 = −9 ✓

⚠️ Common Error

Always find BOTH x AND y. Many students solve for x and forget to substitute back to find y!

⚡ Key Tips for This Unit

✅ Slope = rise/run = (y₂−y₁)/(x₂−x₁). Y's always on top.
✅ y = mx + b: m = slope (steepness), b = y-intercept (starting point).
✅ Function: every x has exactly ONE y. Vertical line test.
✅ To graph: start at b on y-axis, then move rise/run for second point.
