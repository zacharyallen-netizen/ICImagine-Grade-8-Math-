5.1 The Pythagorean Theorem

🔑 THE THEOREM: In a RIGHT triangle, the sum of the squares of the two LEGS equals the square of the HYPOTENUSE.

a² + b² = c²

a, b = legs (shorter sides)    c = hypotenuse (longest side, across from right angle)
⚠️ IMPORTANT: The Pythagorean Theorem ONLY works on RIGHT triangles (triangles with a 90° angle)!

5.2 Finding the Hypotenuse

🔑 WHEN TO USE: When you know BOTH LEGS and need to find the HYPOTENUSE.
Formula: a² + b² = c² → solve for c

Example: Legs are 6 and 8. Find the hypotenuse.

Step 1: Write the formula: a² + b² = c²
Step 2: Substitute: 6² + 8² = c²
Step 3: Square the numbers: 36 + 64 = c²
Step 4: Add: 100 = c²
Step 5: Take square root: √100 = c
Answer: c = 10 ✓

5.3 Finding a Missing Leg

🔑 WHEN TO USE: When you know ONE LEG and the HYPOTENUSE, and need the OTHER LEG.
Rearrange: a² = c² − b²

Example: One leg is 5, hypotenuse is 13. Find the other leg.

Step 1: Write the formula: a² + b² = c²
Step 2: Substitute: 5² + b² = 13²
Step 3: Square: 25 + b² = 169
Step 4: Subtract 25 from both sides: b² = 169 − 25
Step 5: Simplify: b² = 144
Step 6: Take square root: b = √144
Answer: b = 12 ✓
⚠️ REMEMBER: When finding a LEG, you SUBTRACT from the hypotenuse². When finding the HYPOTENUSE, you ADD the two legs².

5.4 Converse of the Pythagorean Theorem

🔑 THE CONVERSE: Given three side lengths a ≤ b ≤ c, plug them into the formula and check: does a² + b² = c²?
If yes → right triangle. If no → not a right triangle.
Always test the two SMALLER sides against the LARGEST.

Example: Sides 5, 12, 13 — Is it a right triangle?

Step 1: Identify largest side: c = 13
Step 2: Test: a² + b² = c²? → 5² + 12² = 13²?
Step 3: Calculate: 25 + 144 = 169
Step 4: Check: c² = 13² = 169
Step 5: Compare: 169 = 169 ✓
YES — it IS a right triangle! ✓

5.6 Distance on the Coordinate Plane

📐 HOW IT WORKS: Connect the two points to form a right triangle. The horizontal distance is Δx = |x₂ − x₁| and the vertical distance is Δy = |y₂ − y₁|. Then use a² + b² = c² where a = Δx, b = Δy, and c = distance.
⚠️ NO DISTANCE FORMULA NEEDED! Just count or subtract to find Δx and Δy, then use the Pythagorean Theorem. That's it!

Example: Find the distance between (1, 2) and (4, 6)

Step 1: Find Δx: |4 − 1| = 3
Step 2: Find Δy: |6 − 2| = 4
Step 3: Use a² + b² = c²: 3² + 4² = c²
Step 4: 9 + 16 = 25 = c²
c = √25 = 5 ✓

📝 Review: Worked Examples & Error Analysis

📝 Ladder: 15 ft ladder, base 9 ft from wall. How high?

Step 1: Ladder = hypotenuse (c = 15), base = leg (b = 9), height = leg (a = ?)
Step 2: Formula: a² + b² = c²
Step 3: Substitute: a² + 81 = 225
Step 4: Subtract 81: a² = 144 (finding a LEG = subtract)
Step 5: Square root: a = √144 = 12 ft ✓

🔍 Spot the Error: Find c: a = 6, b = 8

✗ Student work: 6² + 8² = c² → 36 + 64 = c² → 100 = c² → c = 50
✓ Fix: c² = 100, so c = √100 = 10, NOT 50! Always take the square root at the end.
💡 After adding a² + b², you still need to SQUARE ROOT the result to find c.

🔍 Spot the Error: Find missing leg: c = 13, a = 5

✗ Student work: 5² + b² = 13² → 25 + b² = 169 → b² = 169 + 25 = 194
✓ Fix: When finding a LEG, SUBTRACT: b² = 169 − 25 = 144, so b = 12
💡 Hypotenuse → ADD legs. Missing leg → SUBTRACT from hypotenuse squared.
