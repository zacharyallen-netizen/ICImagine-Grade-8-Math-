📌 For the teacher — how to post a document:
1. Save your file as a PDF using the exact filename shown on the card (for example syllabus.pdf or pacing_guide.pdf).
2. Upload that file into the same folder as the rest of the site (the repository root).
3. That card becomes a live “Download PDF →” button automatically — no code editing needed.

To add a new kind of document, copy one card block in class_information.html and change its title, description, and data-file name.
