Translation Rule: (x, y) → (x + h, y + k)  |  h > 0 → RIGHT  |  h < 0 → LEFT  |  k > 0 → UP  |  k < 0 → DOWN
Remember: Translations are rigid — the image is congruent to the pre-image (same size and shape, just shifted).
Question 1 — Read the Translation
Triangle ABC (blue, left grid) was translated to A′B′C′ (green, right grid). Which rule was used?
A(−3,−2) → A′(1,0): x change = 1−(−3) = +4, y change = 0−(−2) = +2. Rule: (x + 4, y + 2) ✓ (4 right, 2 up)
Question 2
Triangle PQR is translated using rule (x + 7, y − 4). Find P′ (P = (−5, 3)).
(−5+7, 3−4) = (2, −1) = P′(2, −1) ✓
Question 3 — Translate a Rectangle
Rectangle ABCD is translated (x − 2, y + 3). What are the coordinates of B′?
B(4, 1) → (4−2, 1+3) = B′(2, 4) ✓
Question 4
A point moves from (−2, 5) to (6, 1). Write the translation rule.
x: 6−(−2) = +8 → x + 8. y: 1−5 = −4 → y − 4. Rule: (x + 8, y − 4) ✓
Question 5
Triangle P′Q′R′ is the image after the translation (x + 5, y − 3). If Q′ = (4, −1), find the original Q.
Work backwards: Q′ = Q + (5,−3). So Q = Q′ − (5,−3) = (4−5, −1+3) = Q(−1, 2) ✓
Question 6
Translate triangle with vertices (0,0), (3,0), (0,4) by (x − 1, y + 2). What are all three image vertices?
(0,0)→(−1,2), (3,0)→(2,2), (0,4)→(−1,6). Rule: subtract 1 from x, add 2 to y. (−1,2),(2,2),(−1,6) ✓
Question 7
Is a translation a rigid transformation?
Translations slide the figure without changing its size or shape → rigid transformation. Pre-image ≅ Image. ✓
Question 8
Which direction does (x − 3, y + 5) move a figure?
x − 3 → 3 units LEFT (negative h). y + 5 → 5 units UP (positive k). Answer: 3 left, 5 up ✓

Over x-axis: (x, y) → (x, −y)  |  Over y-axis: (x, y) → (−x, y)
Memory trick: the axis you reflect OVER keeps its coordinate. Reflecting over x-axis → x stays, y flips.
Question 1 — Read the Reflection
Rectangle ABCD was reflected. Over which line was it reflected?
A(2,1)→A′(−2,1): x changed sign, y stayed → reflection over y-axis. Rule: (x,y)→(−x,y). ✓
Question 2
Triangle KLM is reflected over the x-axis. Find K′ (K = (7, −4)).
Reflect over x-axis: (x,y)→(x,−y). K(7,−4) → (7,−(−4)) = K′(7, 4) ✓. Only y changes sign.
Question 3
Triangle MNP is reflected over the y-axis. Find M′ (M = (−3, 5)).
Reflect over y-axis: (x,y)→(−x,y). M(−3,5) → (3,5) = M′(3, 5) ✓. Only x changes sign.
Question 4 — Reflect a Triangle
Triangle ABC was reflected over the x-axis. What are the coordinates of C′?
Reflect over x-axis: (x,y)→(x,−y). C(3,5) → C′(3, −5) ✓. The x stays; y changes sign.
Question 5
Triangle PQR is reflected over the x-axis, then that image is reflected over the x-axis again. Find P″ (P = (4, 6)).
P(4,6) → over x-axis → P′(4,−6) → over x-axis again → P″(4,6). Two reflections over the same line = original position! ✓
Question 6
Triangle with vertices A(2,3), B(5,3), C(5,7) is reflected over the y-axis. Find A′, B′, C′.
y-axis reflection → flip x signs: (2,3)→(−2,3), (5,3)→(−5,3), (5,7)→(−5,7). A′(−2,3), B′(−5,3), C′(−5,7) ✓
Question 7
Is a reflection a rigid transformation?
Reflections are rigid transformations (isometries) — size and shape are preserved. Pre-image ≅ Image. ✓
Question 8
A triangle is reflected over the x-axis. The image of vertex T(a, b) is T′(5, −3). Find a and b.
Reflect over x-axis: T(a,b)→T′(a,−b). So T′ = (a,−b) = (5,−3) → a = 5, −b = −3 → b = 3. a = 5, b = 3 ✓

90° CCW: (x,y)→(−y, x)  |  180°: (x,y)→(−x,−y)  |  270° CCW: (x,y)→(y,−x)
90° CW = 270° CCW  |  90° CCW = 270° CW
Question 1 — 90° CCW Rotation
Rectangle ABCD was rotated 90° CCW about the origin. What is the image of vertex B(4, 0)?
90° CCW: (x,y)→(−y,x). B(4,0) → (−0,4) = B′(0, 4) ✓
Question 2
Triangle RST is rotated 180° about the origin. Find R′ (R = (3, −5)).
180°: (x,y)→(−x,−y). R(3,−5) → (−3,−(−5)) = (−3,5) = R′(−3, 5) ✓. Both signs flip.
Question 3
Triangle STU is rotated 270° CCW about the origin. Find S′ (S = (2, 7)).
270° CCW: (x,y)→(y,−x). S(2,7) → (7,−2) = S′(7, −2) ✓
Question 4 — 180° Rotation
Triangle ABC was rotated 180°. If C = (5, 4), what is C′?
180°: (x,y)→(−x,−y). C(5,4) → C′(−5, −4) ✓. Both coordinates change sign.
Question 5
Triangle QRS is rotated 90° CW (= 270° CCW) about the origin. Find Q′ (Q = (−4, 3)).
90° CW = 270° CCW: (x,y)→(y,−x). Q(−4,3) → (3,−(−4)) = (3,4) = Q′(3, 4) ✓
Question 6
Which rotation takes (2, 5) to (−2, −5)?
180°: (x,y)→(−x,−y). (2,5)→(−2,−5) ✓. Both coordinates flip signs → 180°.
Question 7
A triangle is rotated 90° CCW, then rotated another 90° CCW. What single rotation gives the same result?
90° + 90° = 180° total rotation. Adding rotation angles: same as one 180° turn. ✓
Question 8
Is a rotation a rigid transformation?
Rotations are rigid transformations — the figure turns but keeps exact same size and shape. Pre-image ≅ Image. ✓
Score:0/ 8
Dilation: (x, y) → (kx, ky)  |  k > 1 = enlargement  |  0 < k < 1 = reduction  |  Similar figures (SAME shape, DIFFERENT size)
Question 1 — Enlargement k = 2
Rectangle ABCD was dilated with k = 2. What are the coordinates of B′?
Dilation k=2: (x,y)→(2x,2y). B(3,0) → (6,0) = B′(6, 0) ✓. Each coordinate multiplied by 2.
Question 2
Triangle STU is dilated with scale factor k = ½ about the origin. Find T′ (T = (8, −6)).
k = ½: (x,y)→(½x, ½y). T(8,−6) → (4,−3) = T′(4, −3) ✓. k < 1 → reduction.
Question 3
Triangle A(2,4), B(6,4), C(4,8) is dilated with k = 3. Find C′.
k = 3: C(4,8) → (3×4, 3×8) = (12,24) = C′(12, 24) ✓. Enlargement (k > 1).
Question 4
After dilation, image B′ is at (15, −9). The scale factor was k = 3. Find the original B.
Work backwards: B′ = k×B → B = B′/k = (15/3, −9/3) = B(5, −3) ✓
Question 5
A square has side length 4 units. It is dilated with k = 3. What is the new side length? New perimeter? New area?
Side: 4×3=12. Perimeter: 12×4=48 (or 16×3=48). Area: 12²=144 (or 16×3²=144). Area scales by k². ✓
Question 6
Do dilations produce congruent or similar figures?
Dilations change SIZE but preserve SHAPE → similar figures. The only exception: k = 1 gives congruent (no size change). ✓
Question 7
A scale factor of k = 4/5 means the dilation is:
0 < k = 4/5 = 0.8 < 1 → reduction. The image is 80% of the original. ✓
Question 8
Triangle P(1,2), Q(5,2), R(3,6) is dilated k=½. Find all image vertices.
k=½: multiply all coords by ½. P(1,2)→(0.5,1), Q(5,2)→(2.5,1), R(3,6)→(1.5,3). P′(0.5,1), Q′(2.5,1), R′(1.5,3) ✓

Composite: apply transformations ONE AT A TIME in the given order. Order matters!
Congruence vs. Similarity: All rigid (translation + reflection + rotation) → Congruent ≅ · Any dilation → Similar ~
Question 1
Triangle ABC has vertex A(3, 1). Reflect over the y-axis, then translate (x + 2, y − 4). Find the final coordinates of A.
Step 1 reflect over y-axis: A(3,1)→A′(−3,1). Step 2 translate (+2,−4): A′(−3,1)→A″(−3+2,1−4) = (−1,−3) ✓
Question 2
Triangle BCD has vertex B(2, 4). Rotate 90° CCW, then dilate by k = 2. Find the final coordinates of B.
Step 1 rotate 90°CCW: B(2,4)→B′(−4,2). Step 2 dilate k=2: B′(−4,2)→B″(−8,4). (−8, 4) ✓
Question 3
A figure is translated (x+3, y−2), then reflected over the x-axis. Is the final image congruent or similar to the original?
Translation + reflection are both rigid. No dilation → congruent. ✓
Question 4
A figure is reflected over the y-axis, then dilated with k = 3. Congruent or similar?
The dilation (k=3) changes size → result is similar, not congruent. Even one dilation makes the figures only similar. ✓
Question 5
Triangle CDE has vertex C(−2, 5). Rotate 180°, then translate (x − 1, y + 3). Find the final coordinates of C.
Step 1 rotate 180°: C(−2,5)→C′(2,−5). Step 2 translate (−1,+3): C′(2,−5)→C″(1,−2). (1, −2) ✓
Question 6
Reflect (a, b) over the x-axis, then reflect the result over the x-axis again. Where does (a, b) end up?
(a,b)→(a,−b)→(a,−(−b))=(a,b). Two reflections over the same line = original position. ✓
Question 7
Triangle DEF has vertex D(4, −2). Reflect over the y-axis, then reflect over the x-axis. What single transformation has the same effect?
Reflect over y-axis: (4,−2)→(−4,−2). Reflect over x-axis: (−4,−2)→(−4,2). Check 180°: (4,−2)→(−4,2). ✓ Two perpendicular axis reflections = 180° rotation.
Question 8
Figure J undergoes: translation, then rotation 90°CCW, then reflection. Is the final image congruent or similar to Figure J?
Translation, rotation, and reflection are all rigid transformations. No dilation → figures remain congruent. ✓

🔍 Common Mistakes

Can you spot what went wrong?

Rotate (3,−2) by 90° CCW
✗ Student: Student writes: (−2, 3)
✓ Fix: WRONG! Rule is (−y, x). y=−2, so −y = 2. Correct: (2, 3)
💡 Be careful with negatives in rotation rules. −(−2) = +2.
Dilate (4,6) by k=½
✗ Student: Student writes: (2, 8)
✓ Fix: WRONG! ½ × 4 = 2 ✓ but ½ × 6 = 3, not 8. Correct: (2, 3)
💡 Multiply BOTH coordinates by the scale factor.

⚡ Key Tips for This Unit

✅ Translation: (x+a, y+b) — slide. CONGRUENT.
✅ Reflection: x-axis → negate y. y-axis → negate x. CONGRUENT.
✅ Rotation 90° CCW: (−y, x). 180°: (−x,−y). 270° CCW: (y,−x). CONGRUENT.
✅ Dilation: (kx, ky). SIMILAR (not congruent unless k=1).
