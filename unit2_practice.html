⚡ Key Tips for This Unit

✅ Product rule: SAME base → ADD exponents (x³·x⁵ = x⁸)
✅ Quotient rule: SAME base → SUBTRACT exponents (x⁸÷x³ = x⁵)
✅ Power rule: MULTIPLY exponents ((x⁴)³ = x¹²)
✅ Anything to the 0 power = 1 (except 0⁰ which is undefined)
