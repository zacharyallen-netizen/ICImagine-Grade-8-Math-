6.1

Introduction to Functions

Identifying functions across multiple representations, function notation, evaluating functions, and finding domain & range.

Key Concepts — 6.1

What is a Function?

A function is a relation in which every input (x) has exactly one output (y).

  • Inputs can not repeat with different outputs.
  • Outputs can repeat — that's allowed.
Vertical Line Test

If any vertical line crosses the graph in two or more places, it is not a function.

✓ Function
line crosses once
✗ Not a function
line crosses twice
Mapping Diagrams
✓ Function
each input → one output
✗ Not a function
3 maps to two outputs
Function Notation

f(x) = 2x + 4 reads as "f of x".

To evaluate, substitute the input for x using parentheses:

f(3) = 2(3) + 4
f(3) = 6 + 4 = 10
Domain & Range

Domain = all input (x) values
Range = all output (y) values

List values in { } braces. Don't repeat values.

x−2035
y414−2

Domain = {−2, 0, 3, 5}   •   Range = {−2, 1, 4}

6.2

Linear Functions & Slope

Four types of slope, calculating slope from two points and from a graph, identifying slope and y-intercept from y = mx + b, and recognizing linear functions.

Key Concepts — 6.2

Four Types of Slope
Positive
Goes up left to right
m > 0 (e.g., m = 2)
Negative
Goes down left to right
m < 0 (e.g., m = −3)
Zero
Horizontal line
m = 0
Undefined
Vertical line
m = undefined
(not a function)
Slope Formula

Given two points (x₁, y₁) and (x₂, y₂):

m = y₂ − y₁x₂ − x₁   =   riserun

Pick any two clear points. Subtract y's on top, x's on bottom — in the same order.

Through (1, 1) and (4, 3):
m = (3 − 1) ÷ (4 − 1) = 2/3
Rise over Run on a Graph

From (1, 1), rise 2, run 3 → m = 2/3

Slope-Intercept Form
y = mx + b
m = slope — the number multiplied by x
b = y-intercept — the constant; where the line crosses the y-axis (when x = 0)

Example: y = 3x − 5 → slope = 3, y-intercept = −5.

6.3

Writing Equations from Tables

Determining if a table represents a linear function, finding missing values, and writing the equation in slope-intercept form.

Key Concepts — 6.3

Linear or Non-Linear?

A table is linear when the slope Δy ÷ Δx is the same between every pair of consecutive rows.

xyΔy / Δx
02—
15+3 / +1
28+3 / +1
311+3 / +1

✓ Same slope every time → LINEAR (m = 3)

Watch Out for Skipped x's

If the x-values don't go up by 1, you must actually divide, not just look at Δy.

xySlope
−2−10—
0−2+8/+2 = 4
12+4/+1 = 4
414+12/+3 = 4

✓ Slope = 4 every time → LINEAR

Steps: Table → y = mx + b
  1. Find the slope using two rows: m = (y₂ − y₁) ÷ (x₂ − x₁)
  2. Find the y-intercept (b): use the row where x = 0 if it's there. Otherwise, plug your slope and any (x, y) into y = mx + b and solve for b.
  3. Write the equation: y = mx + b
Example: Find equation for table: x: 1, 2, 3, 4   y: 5, 8, 11, 14
m = (8 − 5) ÷ (2 − 1) = 3
x = 0 is missing, so plug in (1, 5):   5 = 3(1) + b   →   b = 2
y = 3x + 2 ✓

6.7

Systems of Equations

What it means to be a solution, predicting solutions by inspection, and solving systems by graphing and substitution.

Key Concepts — 6.7

What is a "Solution"?

A solution to a system of two linear equations is the (x, y) point that makes both equations true at the same time. On a graph, it's where the two lines intersect.

Three Possible Cases (by inspection)
One Solution
Different slopes
Lines cross at one point.
No Solution
Same slope, different b
Parallel lines — never meet.
Infinite Solutions
Same slope, same b
Same line — every point works.
Solving by Substitution
  1. Make sure one equation is solved for a variable (like y = …).
  2. Substitute that expression into the other equation. Now you have one equation in one variable.
  3. Solve for that variable.
  4. Plug your answer back into either equation to find the second variable.
  5. Write your answer as an ordered pair (x, y).
Example:   y = 4x − 1   and   y = 2x + 5
Substitute:   4x − 1 = 2x + 5
2x = 6   →   x = 3
Plug in:   y = 4(3) − 1 = 11
Solution: (3, 11)
