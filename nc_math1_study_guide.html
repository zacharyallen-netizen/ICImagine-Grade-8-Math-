UNIT 1

Equations & Inequalities

🔑 Key Idea Solve equations by undoing operations in reverse order. For inequalities, the rule is the same — except flip the sign whenever you multiply or divide by a negative number.
Variable: a letter for an unknown Equation: two expressions joined by = Inequality: uses <, >, ≤, ≥ Inverse op: the operation that undoes another

📊 Visual: Number line — solution to −2x > 6

📝 Worked Example: 3x + 5 = 20

Step 1 — Subtract 5 from both sides3x = 15
Step 2 — Divide both sides by 3x = 5
✓ Solution: x = 5
⚠️ Inequality flip:
−2x > 6 → divide by −2 → x < −3 (sign flipped!)
🔄 Rearranging formulas:
A = lw → l = A ÷ w

Question 1
Solve: 4x − 7 = 17
Add 7 → 4x = 24. Divide by 4 → x = 6.
Question 2
Solve: −3x ≥ 12  (don't forget the flip!)
Divide by −3 — the sign flips because we divided by a negative. x ≤ −4.
Question 3
Rearrange P = 2l + 2w to solve for w.
Subtract 2l from both sides → P − 2l = 2w. Divide by 2 → w = (P − 2l) / 2. (Note: this equals P/2 − l, but the second option matches the literal step-by-step rearrangement.)
UNIT 2

Linear Functions

🔑 Key Idea A linear function has a constant rate of change. Its graph is a straight line, and its equation can always be written as y = mx + b.
m: slope (rate of change) b: y-intercept (starting value) Slope formula: Δy ÷ Δx Slope-intercept form: y = mx + b

📊 Graph of y = 2x + 1

📝 Slope between two points

Apply formulam = (11 − 3) ÷ (5 − 1)

What is the slope of the line through (2, 1) and (6, 9)?

UNIT 3

Exponential Functions

🔑 Key Idea Exponential functions describe repeated multiplication. Form: y = a · bx where a is the starting value and b is the multiplier.
a: initial value (when x = 0) b: base / multiplier Growth: b > 1 Decay: 0 < b < 1

📊 Growth vs. Decay

📝 Worked Example

Evaluatey = 100 · 2³
Compute power2³ = 8
Multiply100 · 8 = 800
✓ y = 800
🧠 Quick check:
EquationType
y = 5(3)ˣGrowth (b = 3 > 1)
y = 100(0.8)ˣDecay (b = 0.8 < 1)
y = 12(½)ˣDecay (b = 0.5 < 1)
✅ Checkpoint — Unit 3 Score: 0 / 3
Question 1
Is y = 3(0.7)ˣ growth or decay?
b = 0.7 is between 0 and 1 → decay. The output shrinks as x grows.
Question 2
Find y when x = 2 in y = 5(3)ˣ.
3² = 9. Then 5 · 9 = 45.
Question 3
What is the initial value (when x = 0) of y = 12(2)ˣ?
When x = 0, b⁰ = 1, so y = a · 1 = a. Initial value = 12.
UNIT 4

Systems of Equations

🔑 Key Idea A system is two (or more) equations true at the same time. The solution is the point (x, y) that satisfies every equation. Three methods: graphing, substitution, elimination.
System: set of equations solved together Solution: point where lines meet Substitution: plug one equation into the other Elimination: add/subtract to cancel a variable

📊 Two lines, one intersection

📝 Substitution method

Systemy = 2x  |  y = x + 3
Substitute first into second2x = x + 3
Subtract xx = 3
Plug backy = 2(3) = 6
✓ Solution: (3, 6)
📋 How many solutions?
• Lines cross once → 1 solution
• Same line → infinitely many
• Parallel → no solution
✅ Checkpoint — Unit 4 Score: 0 / 3
Question 1
Solve the system: y = 2x and y = x + 3
2x = x + 3 → x = 3 → y = 6. Solution: (3, 6).
Question 2
Solve by elimination: x + y = 7 and x − y = 1
Add the equations: 2x = 8 → x = 4. Then 4 + y = 7 → y = 3. Solution: (4, 3).
Question 3
Two lines are parallel (different y-intercepts). How many solutions does the system have?
Parallel lines never intersect → no solution.
UNIT 5

Statistics

🔑 Key Idea Statistics is about describing data — its center, spread, and any relationships between variables. A scatterplot makes patterns visible.
Mean: sum ÷ count Median: middle value (sorted) Range: max − min Scatterplot: dots on (x, y) axes Correlation: how strongly x and y move together

📊 Three correlation patterns

Look for the direction the cloud of points is leaning.

📝 Worked Example: data set 4, 6, 8, 10, 12

Mean(4 + 6 + 8 + 10 + 12) ÷ 5 = 40 ÷ 5 = 8
Range12 − 4 = 8
Medianmiddle value (sorted) = 8
✓ Mean = 8 · Median = 8 · Range = 8

What is the mean of 5, 10, 15?
(5 + 10 + 15) / 3 = 30 / 3 = 10.

As outdoor temperature increases, ice cream sales increase. What kind of correlation is that?
Both go up together → positive correlation.

UNIT 6

Polynomials

🔑 Key Idea Polynomials are sums of terms with variables and exponents. To simplify: combine like terms (same variable AND same exponent). To multiply two binomials: distribute every term — i.e., FOIL.
Term: single product like 3x² Like terms: same variable + same power Distribute: a(b + c) = ab + ac FOIL: First · Outer · Inner · Last Binomial: 2 terms · Trinomial: 3

📊 FOIL: (x + 3)(x + 4)

Sum: x² + 4x + 3x + 12 → combine like terms → x² + 7x + 12

📝 Combining Like Terms

Original3x + 5x + 2y − y
Group like(3x + 5x) + (2y − y)
Combine8x + y
✓ 8x + y
📋 Distribute:
2(x + 5) = 2·x + 2·5 = 2x + 10
−3(2x − 4) = −6x + 12
⚠️ Watch the negative — distribute the sign too.
✅ Checkpoint — Unit 6 Score: 0 / 3
Question 1
Combine: 4x² + 2x − 3x² + 5x
(4x² − 3x²) + (2x + 5x) = x² + 7x.
Question 2
Distribute: −3(2x − 4)
−3 · 2x = −6x. −3 · (−4) = +12. Result: −6x + 12.
Question 3
FOIL: (x + 5)(x − 2)

UNIT 7

Quadratics & Factoring

🔑 Key Idea Standard form: ax² + bx + c. Factoring rewrites a quadratic as a product of two binomials, which lets you solve = 0 by setting each factor to zero.
Quadratic: degree-2 polynomial GCF: greatest common factor Trinomial: 3 terms Difference of squares: a² − b² Perfect square trinomial: (a + b)² Zero Product: if AB = 0, then A=0 or B=0

🧠 The Four Methods

1️⃣ Simple Trinomial (a = 1)

x² + 5x + 6Find two numbers that multiply to 6 and add to 5
→ 2 and 32 · 3 = 6 ✓   2 + 3 = 5 ✓
✓ (x + 2)(x + 3)

2️⃣ AC Method (a ≠ 1)

2x² + 7x + 3a · c = 2 · 3 = 6
Find numbersmultiply to 6, add to 7 → 6 and 1
Rewrite + group2x² + 6x + x + 3 → 2x(x+3) + 1(x+3)
✓ (2x + 1)(x + 3)

3️⃣ Difference of Squares

Patterna² − b² = (a − b)(a + b)
Examplex² − 9 = x² − 3²
✓ (x − 3)(x + 3)

4️⃣ Perfect Square Trinomial

x² + 6x + 9√x² = x · √9 = 3
Check middle2 · x · 3 = 6x ✓
✓ (x + 3)²

🎯 Solving by Factoring

Equationx² − 5x + 6 = 0
Factor(x − 2)(x − 3) = 0
Zero Product Propertyx − 2 = 0  OR  x − 3 = 0
✓ Solutions: x = 2 and x = 3
