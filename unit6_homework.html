Key Rules: Each input can have only ONE output. Use the Vertical Line Test for graphs. Use input uniqueness for tables and ordered pairs.
Part A — Is It a Function?
Question 1
The table shows inputs and outputs. Is this relation a function?
x1234
y581114
Each input (x) maps to exactly one output (y). No x value repeats → this IS a function. ✓ (It's actually linear: y = 3x + 2)
Question 2
The mapping diagram shows: input 4 → outputs 7 AND 9. Is this relation a function?
If ONE input has TWO or more outputs, it is NOT a function. Input 4 → {7, 9} violates the function rule. ✓
Question 3
Apply the Vertical Line Test to the graph below. Is this a function?
A circle fails the Vertical Line Test — vertical lines in the interior cross the circle at TWO points (top and bottom). NOT a function. ✓
Question 4
Which set of ordered pairs represents a function?
{(0,4),(2,4),(3,4),(5,4)}: every x is different → function ✓. It's a horizontal line y=4. Note: outputs CAN repeat — only INPUTS must be unique! ✓
Question 5
A vertical line x = 3 touches a graph at only one point: (3, 7). What can you conclude?
At x = 3 there is exactly one y-value (7) — VLT passes there. We'd need to check ALL vertical lines to confirm the whole graph is a function. ✓
Part B — Function Notation
Question 6
Given f(x) = 5x − 3, find f(4).
f(4) = 5(4) − 3 = 20 − 3 = 17 ✓. Substitute 4 everywhere you see x.
Question 7
Given g(x) = −2x + 10, find g(3).
g(3) = −2(3) + 10 = −6 + 10 = 4 ✓
Question 8
The function h(x) = 4x. What is h(−2)?
h(−2) = 4(−2) = −8 ✓. Don't forget to keep the negative sign when substituting.
Question 9
Domain and Range: The function f = {{(2,5),(4,9),(6,13),(8,17)}}. What is the range?
The range = all output (y) values = {{5, 9, 13, 17}}. The domain = {{2, 4, 6, 8}} (inputs). ✓
Question 10
Which equation is NOT a function?
x = 4 is a vertical line — it assigns infinitely many y-values to the same x. It fails the VLT and is NOT a function. All the others assign exactly one y per x. ✓

Compare: Find slope (m) and y-intercept (b) for each function. From a graph: slope = rise/run. From a table: slope = Δy/Δx.
Part A — Rates of Change
Question 1
Function A is shown in the graph. Function B has equation y = 2x + 1. Which function has a greater rate of change?
From the graph, Function A rises 3 units for every 1 unit right → slope = 3. Function B has slope 2. Since 3 > 2, Function A has a greater rate of change. ✓
Question 2
Function P: y = 4x − 3. Function Q from table: x=0→y=1, x=1→y=5, x=2→y=9, x=3→y=13. Which has the greater rate of change?
Function P: slope = 4. Function Q: Δy/Δx = (5−1)/(1−0) = 4/1 = 4. Both have slope 4 — same rate of change, different y-intercepts (P: −3, Q: +1). ✓
Question 3
Function A: y = x + 4 (y-intercept = 4). Function B: y = 3x − 2 (y-intercept = −2). At what x-value do they intersect? (Where are they equal?)
Set equal: x + 4 = 3x − 2 → 6 = 2x → x = 3. Check: y = 3+4=7 and y=3(3)−2=7 ✓
Question 4
Taxi A: y = 2.50x + 3 (x = miles). Taxi B: y = 1.50x + 7. For a 5-mile ride, which taxi costs MORE?
Taxi A at x=5: 2.50(5)+3=12.50+3=$15.50. Taxi B: 1.50(5)+7=7.50+7=$14.50. Taxi A costs more for 5 miles. ✓
Part B — Linear vs. Nonlinear
Question 5
Table: x=0→y=3, x=1→y=5, x=2→y=7, x=3→y=9. Is this linear?
Check: 5−3=2, 7−5=2, 9−7=2. Constant Δy = 2 → linear. Equation: y = 2x + 3. ✓
Question 6
Table: x=0→y=1, x=1→y=2, x=2→y=4, x=3→y=8. Is this linear?
Δy: 2−1=1, 4−2=2, 8−4=4. The differences INCREASE — not constant → nonlinear. This is exponential: y = 2ˣ ✓
Question 7
Which equation is linear?
A linear equation has the form y = mx + b with no exponents on x. Only y = 3x − 7 fits (m=3, b=−7). The others have x², 2ˣ, or 1/x which are nonlinear. ✓
Question 8
Write the equation of the line that passes through (0, 5) and (3, 14).
y-intercept b = 5 (given: passes through (0,5)). Slope m = (14−5)/(3−0) = 9/3 = 3. Equation: y = 3x + 5 ✓. Check: at x=3: 3(3)+5=14 ✓

Slope formula: m = (y₂ − y₁) / (x₂ − x₁)  |  y-intercept: value of y when x = 0
Interpreting Linear Models
Question 1
The line y = 45x represents a car traveling at constant speed (x = hours, y = miles). What does the slope 45 represent?
Slope = rate of change = 45 miles per 1 hour = 45 mph. The car covers 45 miles for every 1 hour of driving. ✓
Question 2
Phone plan: y = 0.10x + 25 (x = minutes, y = monthly cost in $). What is the monthly base charge?
The y-intercept (b = 25) is the cost when x = 0 minutes → the base monthly charge is $25. The slope 0.10 means $0.10 per additional minute. ✓
Question 3
A plumber charges y = 60x + 50 (x = hours). What is the cost for a 3-hour job?
y = 60(3) + 50 = 180 + 50 = $230 ✓. The $50 is the service call fee; $60/hr is the hourly rate.
Question 4
Find the slope of the line through (2, 7) and (6, 19).
m = (19−7)/(6−2) = 12/4 = 3 ✓. Rise = 12, Run = 4. Check: starting from (2,7), go right 4 → 3×4=12 → 7+12=19 ✓
Question 5
The y-intercept of a line is −4 and slope is 2. Write the equation.
Slope-intercept form: y = mx + b. m = 2, b = −4 → y = 2x − 4 ✓
Question 6
Write the equation of a line through (0, 6) and (4, 14).
b = 6 (y-intercept is given). m = (14−6)/(4−0) = 8/4 = 2. Equation: y = 2x + 6 ✓
Question 7
A candle is 12 inches tall and burns at 1.5 inches per hour. Which equation models the candle height y after x hours?
Candle DECREASES in height: slope = −1.5 (negative). Starts at 12 in → b = 12. Equation: y = −1.5x + 12 ✓
Question 8
After how many hours does the candle from Q7 go out (height = 0)?
Set y = 0: 0 = −1.5x + 12 → 1.5x = 12 → x = 8 hours ✓

Key terms: Increasing (going UP left to right) · Decreasing (going DOWN) · Constant (flat/horizontal) · Maximum (highest point) · Minimum (lowest point)
Reading Qualitative Graphs
Question 1
The graph below shows distance over time during a walk. How would you describe the middle segment?
The middle portion of the graph is horizontal (constant) — the distance doesn't change, meaning the person stopped walking for a period. ✓
Question 2
The graph shows a roller coaster height over time. Describe what's happening at the peak.
The peak represents the maximum — the highest point. Before the peak, the graph is increasing; after, it is decreasing. ✓
Question 3
A graph is increasing from x = 0 to x = 4, constant from x = 4 to x = 6, then decreasing from x = 6 to x = 10. This could represent which scenario?
Increase → constant → decrease matches: car speeds up → holds constant speed → slows down (brakes). ✓
Question 4
On a graph of profit vs. time, a section is decreasing. What does this mean in context?
Decreasing on a profit-vs-time graph means profit is going DOWN as time passes. The company is losing money during this interval. ✓
Question 5
A graph of water temperature vs. time shows: fast increase, then slow increase, then constant. Which graph feature appears near the end?
The water heats quickly, then slowly, then reaches a boiling/steady temperature → constant plateau = maximum temperature reached ✓
Question 6
What is the key difference between a graph that is decreasing and one that is constant?
Decreasing = y-values get smaller from left to right (negative slope). Constant = y-values stay the same (slope = 0, horizontal line). ✓
Question 7
A student says a constant section of a graph means "nothing is happening." Is this correct?
Constant does NOT mean zero! It means the value isn't changing at that moment. Example: driving 60 mph — distance increases at a constant rate, but the distance-vs-time graph segment IS increasing. A horizontal distance graph would mean you're stopped. Context matters! ✓
Question 8
A ball is thrown upward. The height-vs-time graph is: increases, reaches a maximum, then decreases. What happens to the rate of change (slope) as the ball goes up?
As the ball goes up, it slows down due to gravity. The slope is positive but decreasing toward zero at the peak. After the peak, slope becomes negative (ball falls). ✓

🔍 Common Mistakes

Can you spot what went wrong?

Slope: (2,3) and (6,11)
✗ Student: Student writes: m = (6−2)/(11−3) = 4/8 = ½
✓ Fix: WRONG! Swapped x and y. Numerator is Y's, denominator is X's. m = 8/4 = 2
💡 Slope = RISE/RUN = (y₂−y₁)/(x₂−x₁). Y's on top!
y = mx + b: m=3, b=−2
✗ Student: Student writes: y = −2x + 3
✓ Fix: WRONG! Swapped slope and intercept. Correct: y = 3x − 2
💡 m goes with x (slope). b is the constant (y-intercept).

⚡ Key Tips for This Unit

✅ Slope = rise/run = (y₂−y₁)/(x₂−x₁). Y's always on top.
✅ y = mx + b: m = slope (steepness), b = y-intercept (starting point).
✅ Function: every x has exactly ONE y. Vertical line test.
✅ To graph: start at b on y-axis, then move rise/run for second point.
