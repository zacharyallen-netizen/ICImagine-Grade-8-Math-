Triangle Theorems
💡 Interior Sum: All 3 angles = 180° | Exterior Angle: = Sum of the 2 remote interior angles
Question 1
Find the missing angle x:
65 + 80 + x = 180 → x = 35°
Question 2
Find the missing angle x:
40 + 90 + x = 180 → x = 50°
Question 3
Find the missing angle x:
x + 53 + 55 = 180 → x = 72°
Question 4
Find the missing angle x:
30 + x + 90 = 180 → x = 60°
Question 5
Find x (exterior angle):
Exterior = 45 + 75 = 120° (sum of remote interior)
Question 6
Find the missing interior angle x:
Exterior angle = sum of non-adjacent interior angles. 135 = 55 + x, so x = 135 − 55 = 80°
Question 7
Find the missing interior angle x:
Exterior angle = sum of non-adjacent interior angles. 118 = 46 + x, so x = 118 − 46 = 72°
Question 8
Find the missing angle x:
50 + 50 + x = 180 → x = 80° (isosceles triangle!)

🔍 Common Mistakes

Can you spot what went wrong?

Co-interior angles: given 75°
✗ Student: Student writes: x = 75° (because angles are equal)
✓ Fix: WRONG! Co-interior angles are SUPPLEMENTARY (= 180°). x = 105°
💡 Co-interior = same-side interior = 180°. Not equal!
Triangle: 65° + 80° + x = 180°
✗ Student: Student writes: x = 65°
✓ Fix: WRONG! 180 − 145 = 35°, not 65°. Subtraction error!
💡 Double-check arithmetic on every problem.

⚡ Key Tips for This Unit

✅ Complementary = 90° (C for Corner). Supplementary = 180° (S for Straight).
✅ Vertical angles are ALWAYS equal.
✅ Exterior angle = sum of the two remote interior angles.
✅ All triangle angles ALWAYS sum to exactly 180°.
