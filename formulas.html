📋 Formula Reference Sheet

Tap any formula to expand with examples · All 8th Grade NC Standards

📊
Rational Numbers U1
a/b (fraction)
▼

Any number that can be written as a/b

Example: 3/4 = 0.75 (terminating), 1/3 = 0.333... (repeating). Both are rational!
💡 Repeating decimals → use the 9's rule: 0.27̄ = 27/99 = 3/11
📊
Perfect Squares U1
√n = integer if n is perfect square
▼

Square roots of perfect squares are rational

Example: √49 = 7, √144 = 12, √1 = 1, √0 = 0
💡 Memorize: 1,4,9,16,25,36,49,64,81,100,121,144
💥
Product Rule U2
x^a · x^b = x^(a+b)
▼

Multiply same base → add exponents

Example: x³ · x⁵ = x⁸, 2⁴ · 2³ = 2⁷ = 128
💡 Only works with the SAME base!
💥
Quotient Rule U2
x^a ÷ x^b = x^(a−b)
▼

Divide same base → subtract exponents

Example: x⁸ ÷ x³ = x⁵, 5⁶ ÷ 5² = 5⁴
💡 If exponents are equal: x⁰ = 1
💥
Power Rule U2
(x^a)^b = x^(a·b)
▼

Power to a power → multiply exponents

Example: (y⁴)³ = y¹², (2³)² = 2⁶ = 64
💡 Distribute to coefficients too: (3x²)³ = 27x⁶
💥
Scientific Notation U2
a × 10^n where 1 ≤ a < 10
▼

Standard form for very large/small numbers

Example: 45,000,000 = 4.5 × 10⁷, 0.00032 = 3.2 × 10⁻⁴
💡 Multiply: multiply a's, add n's. Divide: divide a's, subtract n's.
🔢
Solving Equations U3
ax + b = c → x = (c−b)/a
▼

Isolate the variable using inverse operations

Example: 3x + 7 = 22 → 3x = 15 → x = 5
💡 Special: 5 = 5 → infinite solutions. 3 = 7 → no solution.
🔢
Inequalities U3
ax + b > c (flip if a < 0)
▼

Like equations, but FLIP when × or ÷ by negative

Example: -2x > 6 → x < -3 (FLIP!)
💡 Open circle for < >. Closed circle for ≤ ≥.
📐
Angle Pairs U4
Supp = 180° | Comp = 90° | Vert = equal
▼

Supplementary, Complementary, Vertical, Linear

Example: If one angle is 65°: supplement = 115°, complement = 25°
💡 Vertical angles are ALWAYS equal. Linear pairs ALWAYS = 180°.
📐
Parallel Lines U4
Corr = Alt Int = Alt Ext = EQUAL
▼

Transversal creates equal and supplementary angles

Example: Corresponding: same position at each intersection → equal
💡 Co-interior (same-side) angles = 180° (supplementary)
📐
Triangle Theorems U4
Interior sum = 180° | Exterior = sum of remotes
▼

Interior sum and exterior angle

Example: Angles 65° + 80° + x = 180° → x = 35°. Exterior = 65° + 80° = 145°
💡 Exterior angle is ALWAYS larger than either remote interior angle.
📏
Pythagorean Theorem U5
a² + b² = c²
▼

Right triangle side relationship

Example: Legs 3,4 → c = √(9+16) = √25 = 5. Missing leg: a = √(c²−b²)
💡 c is ALWAYS the hypotenuse (longest side, across from the right angle).
📏
Distance Formula U5
d = √((x₂−x₁)² + (y₂−y₁)²)
▼

Pythagorean Theorem on a grid

Example: (1,2) to (4,6): d = √(9+16) = √25 = 5
💡 It IS the Pythagorean Theorem — horizontal = a, vertical = b, distance = c.
📈
Slope U6
m = (y₂−y₁)/(x₂−x₁) = rise/run
▼

Rate of change between two points

Example: (1,2) and (3,8): m = (8−2)/(3−1) = 6/2 = 3
💡 Positive = uphill, Negative = downhill, Zero = flat, Undefined = vertical
📈
Slope-Intercept Form U6
y = mx + b
▼

Linear equation with slope and y-intercept

Example: m = slope (rate of change), b = y-intercept (starting value)
💡 To graph: plot b on y-axis, then use m as rise/run.
🔬
Line of Best Fit U7
y = mx + b (approx.)
▼

Trend line through scatterplot data

Example: Approx. equal points above and below. Use to PREDICT values.
💡 Does NOT need to pass through actual data points.
🔬
Relative Frequency U7
RF = cell ÷ total
▼

Part ÷ Total = proportion

Example: Joint: 45/120 = 37.5%. Conditional: 45/60 = 75% (of that row/column)
💡 Joint = ÷ grand total. Conditional = ÷ row or column total.
🔄
Translation U8
(x,y) → (x+a, y+b)
▼

Slide every point same distance

Example: (2,3) translated by (+4,−1) → (6,2)
💡 CONGRUENT — size and shape don't change.
🔄
Reflection U8
x-axis: (x,y)→(x,−y) | y-axis: (x,y)→(−x,y)
▼

Flip over a line

Example: (3,5) over x-axis → (3,−5). Over y-axis → (−3,5)
💡 CONGRUENT — mirror image.
🔄
Rotation U8
90°CCW:(−y,x) | 180°:(−x,−y) | 270°CCW:(y,−x)
▼

Turn around origin

Example: (2,5) rotated 90° CCW → (−5,2)
💡 CONGRUENT — same size, different orientation.
🔄
Dilation U8
(x,y) → (kx, ky)
▼

Scale by factor k

Example: k=2: (3,4)→(6,8) enlarged. k=0.5: (3,4)→(1.5,2) reduced.
💡 SIMILAR (not congruent unless k=1). Angles stay same, sides proportional.
📦
Cylinder Volume VOL
V = πr²h
▼

Stack of circles

Example: r=5, h=8: V = π(25)(8) = 200π ≈ 628.3
💡 Base area × height. Watch for diameter vs radius!
📦
Cone Volume VOL
V = ⅓πr²h
▼

⅓ of a cylinder

Example: r=3, h=12: V = ⅓π(9)(12) = 36π ≈ 113.1
💡 3 cones = 1 cylinder with same r and h.
📦
Sphere Volume VOL
V = ⁴⁄₃πr³
▼

Only needs radius

Example: r=6: V = ⁴⁄₃π(216) = 288π ≈ 904.8
💡 r CUBED not squared! No height needed.
