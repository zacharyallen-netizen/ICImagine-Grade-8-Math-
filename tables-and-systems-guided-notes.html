Writing Linear Equations from a Table

The Big Idea

Every linear equation can be written as y = mx + b. To go from a table to an equation, we need to find two things:

  • m = the slope (the rate of change)
  • b = the y-intercept (the y-value when x = 0)
y = mx + b
Quick check: is the table linear?
A table shows a linear relationship only when the y-values change by the same amount for equal steps in x. If x goes up by 1 each row and y jumps around, it's not linear.

Finding the Slope from a Table

m = (y₂ − y₁) ÷ (x₂ − x₁) = change in y ÷ change in x

Pick any two rows of the table. Subtract to get the change in y and the change in x. Divide.

Solving Systems by Graphing

What Is a System of Equations?

A system of equations is two or more equations that share the same variables. In 8th grade, we work with systems of two linear equations in two variables (x and y).

EQ 1y = 2x + 1
EQ 2y = −x + 4

Graphing Method: The Big Idea

Graph each line on the same coordinate plane. The point where the two lines intersect is the solution.

solution of a system = intersection point (x, y)
Steps
  1. Graph the first line (use slope and y-intercept, or a table).
  2. Graph the second line on the same plane.
  3. Find where they cross. That ordered pair is the solution.
  4. Check by plugging the point into both equations.
EXAMPLE Solve by graphing
EQ 1y = 2x + 1
EQ 2y = −x + 4
  1. EQ 1: slope 2, y-int 1. Graph from (0, 1): right 1, up 2 → (1, 3), (2, 5)...
  2. EQ 2: slope −1, y-int 4. Graph from (0, 4): right 1, down 1 → (1, 3), (2, 2)...
  3. Read the intersection: (1, 3)
  4. Check. EQ 1: 3 = 2(1) + 1 = 3 ✓. EQ 2: 3 = −(1) + 4 = 3 ✓. Both true. Solution: (1, 3).

Three Possible Outcomes

When you graph a system of two lines, one of three things will happen:

Case 1 · One Solution
The two lines cross at exactly one point. The system has one solution — that point. (Different slopes.)
Case 2 · No Solution
The two lines are parallel. They never cross. The system has no solution. (Same slope, different y-intercepts.)
Case 3 · Infinitely Many Solutions
The two equations are the same line. Every point on the line works. Infinitely many solutions. (Same slope AND same y-intercept.)
Try It · Read the Graph

Read the graph: the two lines cross at the red point. Enter the solution as an ordered pair.

Solution: (3, 2)
Check EQ 1:   y = x − 1 → 2 = 3 − 1 = 2 ✓
Check EQ 2:   y = −2x + 8 → 2 = −2(3) + 8 = 2 ✓

What Does "Solution" Actually Mean?

The Core Definition

A solution to a system of equations is an ordered pair (x, y) that makes every equation in the system true at the same time.

Two ways to say it
  • Graphically: the solution is the point where the lines cross.
  • Algebraically: the solution is the (x, y) that, when plugged in, satisfies both equations.

Testing Whether a Point Is a Solution

To test a candidate point (a, b), plug x = a and y = b into each equation. If the point makes both equations true, it's a solution. If it fails even one, it's not a solution.

EXAMPLE Is (2, 5) a solution?
EQ 1y = 2x + 1
EQ 2y = 3x − 1
  1. Test EQ 1 with x = 2, y = 5:
        5 = 2(2) + 1 = 5 ✓ True.
  2. Test EQ 2 with x = 2, y = 5:
        5 = 3(2) − 1 = 5 ✓ True.
  3. Both equations are true, so (2, 5) is a solution.
EXAMPLE Is (4, 3) a solution?
EQ 1y = x − 1
EQ 2y = 2x − 5
  1. Test EQ 1 with x = 4, y = 3:
        3 = 4 − 1 = 3 ✓ True.
  2. Test EQ 2 with x = 4, y = 3:
        3 = 2(4) − 5 = 3 ✓ True.
  3. Both true, so (4, 3) is a solution. Even though it looked suspicious, the test confirms it.
Takeaway
You cannot tell by eyeballing. Always test both equations.
Try It · Test the Point
EQ 1y = 3x − 2
EQ 2y = −x + 6

Is (2, 4) a solution to this system?

Yes — (2, 4) is a solution.
EQ 1: 4 = 3(2) − 2 = 4 ✓
EQ 2: 4 = −(2) + 6 = 4 ✓
Both true.
Try It · Test the Point (2)
EQ 1y = 2x + 3
EQ 2y = x + 5

Is (1, 6) a solution to this system?

No — (1, 6) is not a solution.
EQ 1: 6 = 2(1) + 3 = 5.   6 ≠ 5 ✗ (fails EQ 1)
EQ 2: 6 = 1 + 5 = 6 ✓
A point must satisfy both to be a solution. The real solution here is (2, 7).

Solving Systems by Substitution

Why Substitution?

Graphing is great, but it's slow and it's hard to read an exact answer when the solution isn't a clean integer like (1, 3). Substitution is an algebraic shortcut that always gives an exact answer.

The Steps

Substitution Procedure
  1. Make sure one equation is solved for a variable (usually y = ...).
  2. Substitute the expression into the other equation.
  3. Solve that equation for x.
  4. Plug x back in to find y.
  5. Write the solution as an ordered pair (x, y) and check.
EXAMPLE 1 Both equations in y = form
EQ 1y = 2x + 3
EQ 2y = −x + 6
  1. Both equations are solved for y. Since they both equal y, they must equal each other. Set them equal: 2x + 3 = −x + 6
  2. Solve for x. Add x to both sides: 3x + 3 = 6. Subtract 3: 3x = 3. Divide: x = 1.
  3. Back-substitute into either equation. Using EQ 1: y = 2(1) + 3 = 5.
  4. Solution: (1, 5)
  5. Check in EQ 2: 5 = −(1) + 6 = 5 ✓
EXAMPLE 2 One equation in y = form, one not
EQ 1y = 3x − 1
EQ 22x + y = 9
  1. EQ 1 is already solved for y. So y = 3x − 1.
  2. Substitute (3x − 1) in place of y in EQ 2: 2x + (3x − 1) = 9
  3. Solve for x. Combine like terms: 5x − 1 = 9. Add 1: 5x = 10. Divide: x = 2.
  4. Back-substitute into EQ 1: y = 3(2) − 1 = 5.
  5. Solution: (2, 5)
  6. Check EQ 2: 2(2) + 5 = 9 ✓
EXAMPLE 3 Quick rearrange required
EQ 1x + y = 7
EQ 2y = 2x + 1
  1. EQ 2 is already solved for y.
  2. Substitute (2x + 1) for y in EQ 1: x + (2x + 1) = 7
  3. Solve: 3x + 1 = 7 → 3x = 6 → x = 2.
  4. Back-substitute: y = 2(2) + 1 = 5.
  5. Solution: (2, 5)
Note
If neither equation is solved for a variable, pick the easier one and solve it first. If you see a coefficient of 1 on x or y, that equation is easiest.
Try It · Substitution Practice 1
EQ 1y = x + 4
EQ 2y = 3x − 2

Solve the system using substitution. Enter the solution as (x, y).

Solution: (3, 7)
Set equal: x + 4 = 3x − 2
4 + 2 = 3x − x → 6 = 2x → x = 3
y = 3 + 4 = 7
Try It · Substitution Practice 2
EQ 1y = 2x − 3
EQ 2x + y = 9

Solve using substitution. (EQ 1 is already done for you — sub into EQ 2.)

Solution: (4, 5)
Sub (2x − 3) for y in EQ 2: x + (2x − 3) = 9
3x − 3 = 9 → 3x = 12 → x = 4
y = 2(4) − 3 = 5
Try It · Substitution Practice 3 (quick rearrange)
EQ 12x + y = 10
EQ 2y = x + 1

EQ 2 is solved for y — sub (x + 1) for y into EQ 1.

Solution: (3, 4)
Sub (x + 1) for y in EQ 1: 2x + (x + 1) = 10
3x + 1 = 10 → 3x = 9 → x = 3
y = 3 + 1 = 4

Graphing vs. Substitution — When to Use Which

Graphing
Best for visualizing the solution and checking whether the lines intersect, are parallel, or are the same. Limited when the answer isn't a whole number.
Substitution
Best when at least one equation is already solved for x or y (or is easy to solve). Always gives an exact answer — fractions and all.
