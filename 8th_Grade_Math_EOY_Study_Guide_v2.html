UNIT 2

Exponents & Scientific Notation

NC.8.EE.1 · NC.8.EE.3 · NC.8.EE.4

You Need to Know

  • Evaluate expressions with positive integer exponents
  • Product rule: same base → add exponents (x³ · x⁴ = x⁷)
  • Quotient rule: same base → subtract exponents
  • Power of a power: multiply exponents ((x³)⁴ = x¹²)
  • Zero & negative exponents (x⁰ = 1; x⁻² = 1/x²)
  • Convert standard ↔ scientific notation
  • Multiply & divide numbers in scientific notation

Exponent Rules at a Glance

  • Product: xᵃ · xᵇ = xa+b
  • Quotient: xᵃ ÷ xᵇ = xa−b
  • Power: (xᵃ)ᵇ = xab
  • Zero: x⁰ = 1 (x ≠ 0)
  • Negative: x⁻ⁿ = 1/xⁿ
  • Sci. Notation: a × 10ⁿ  where 1 ≤ a < 10

UNIT 3

Solving Linear Equations

NC.8.EE.7

You Need to Know

  • Solve one-step & two-step equations
  • Use the distributive property in multi-step equations
  • Combine like terms on one or both sides
  • Solve equations with variables on both sides
  • Identify: one solution / no solution / infinitely many
  • Translate word problems into equations & solve

Solution Types

  • One solution: x = a number → lines intersect
  • No solution: false (3 = 7) → parallel lines
  • Infinitely many: always true (5 = 5) → same line
  • Steps: Distribute → Combine → Move variables → Isolate

UNIT 4

Angles, Lines & Triangles

NC.8.G.5

You Need to Know

  • Identify complementary, supplementary, vertical & adjacent angles
  • Find unknown angle measures from relationships
  • Name angle pairs formed by a transversal cutting parallel lines
  • Find angle measures with parallel lines & transversals
  • Triangle Angle-Sum Theorem: interior angles add to 180°
  • Exterior Angle Theorem: exterior = sum of two non-adjacent interior angles

Quick Facts

  • Complementary: two angles sum to 90°
  • Supplementary: two angles sum to 180°
  • Vertical angles: always equal (not 90° unless specified)
  • Corresponding angles: equal (parallel lines)
  • Alternate interior/exterior: equal (parallel lines)
  • Same-side interior: supplementary (sum to 180°)

UNIT 5

Pythagorean Theorem

NC.8.G.6 · NC.8.G.7 · NC.8.G.8

You Need to Know

  • Identify the legs and hypotenuse of a right triangle
  • Use a² + b² = c² to find the hypotenuse
  • Rearrange to find a missing leg: a² = c² − b²
  • Use the Converse to check if a triangle is a right triangle
  • Solve real-world word problems with the theorem
  • Find the distance between two coordinate points

Key Formulas

  • a² + b² = c²  (c = hypotenuse, always longest)
  • Converse: if a²+b²=c² → it IS a right triangle
  • Distance: d = √[(x₂−x₁)² + (y₂−y₁)²]
  • Pythagorean triples: 3-4-5, 5-12-13, 8-15-17
  • Multiples work too: 6-8-10, 9-12-15 …

UNIT 6

Functions, Linear Relationships & Systems

NC.8.F.1–5 · NC.8.EE.5 · NC.8.EE.6 · NC.8.EE.8

You Need to Know

  • Define a function; use the vertical line test
  • Identify functions from tables, graphs, ordered pairs
  • Find domain and range
  • Tell linear from nonlinear (graphs, tables, equations)
  • Calculate slope from a graph, table, or two points
  • Write equations in slope-intercept form: y = mx + b
  • Graph lines from slope-intercept form
  • Solve systems by graphing; identify solution type

Key Formulas

  • Slope: m = rise/run = (y₂−y₁)/(x₂−x₁)
  • Slope-intercept: y = mx + b
  • b = y-intercept (where line crosses y-axis)
  • One solution: lines intersect at one point
  • No solution: parallel (same slope, different b)
  • Infinite solutions: same line (same m and b)

UNIT 7

Bivariate Data & Statistics

NC.8.SP.1 · NC.8.SP.2 · NC.8.SP.3 · NC.8.SP.4

You Need to Know

  • Construct and interpret scatter plots
  • Describe association: positive / negative / none, linear / nonlinear
  • Identify outliers and clusters in data
  • Draw a line of best fit; write its equation
  • Use the line of best fit to make predictions
  • Build and read two-way frequency tables
  • Calculate relative & joint frequencies

Association Types

  • Positive: as x ↑, y ↑ (dots go up-right)
  • Negative: as x ↑, y ↓ (dots go down-right)
  • No association: no clear direction
  • Linear: data follows a straight line pattern
  • Relative frequency: part ÷ total (expressed as %)

UNIT 8

Transformations, Congruence & Similarity

NC.8.G.1 · NC.8.G.2 · NC.8.G.3 · NC.8.G.4

You Need to Know

  • Identify translations, reflections, rotations, and dilations
  • Translate a figure and write the coordinate rule
  • Reflect across the x-axis, y-axis, or the line y = x
  • Rotate 90°, 180°, or 270° about the origin (CW and CCW)
  • Rigid motions (translate, reflect, rotate) → congruent figures
  • Describe a sequence of transformations between two figures
  • Perform dilations; find and apply the scale factor
  • Dilations → similar figures (same shape, different size)

Coordinate Rules

  • Translation: (x, y) → (x+a, y+b)
  • Reflect over x-axis: (x, y) → (x, −y)
  • Reflect over y-axis: (x, y) → (−x, y)
  • Rotate 90° CCW: (x, y) → (−y, x)
  • Rotate 180°: (x, y) → (−x, −y)
  • Rotate 270° CCW (=90° CW): (x, y) → (y, −x)
  • Dilation by k: (x, y) → (kx, ky)
